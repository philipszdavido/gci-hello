# GCI-Hello World

An npm module that logs `Hello World!` to the console.

## Installation
```sh
npm install gci-hello --save
```

## Usage
```js
const helloworld = require('gci-hello');
    
helloworld(); // Logs "Hello World!"
```
## Why

A small fun bit of practice for the [2017 Google Code-In][1]!
