# GCI-hello
---

## What

A small npm module that logs `Hello World!` to the console when run.

## Usage

`npm install --save gci-hello`

    const hw = require('gci-hello')
    
    hw() // Logs "Hello World!"
    
## Why

A small fun bit of practice for the [2017 Google Code-In][1]!
